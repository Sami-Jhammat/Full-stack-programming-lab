# Lab_02 — JavaScript Interactivity

This folder contains solutions for Lab 02 tasks (HTML + CSS + JavaScript).

## How to run
1. Open `Lab_02-JS-Interactivity/index.html` in your browser (double-click).
2. Or open each task folder and run its `index.html`.

## Tasks included
- Task 1: Dynamic Quiz
- Task 2: Interactive Calculator
- Task 3: Simple To-Do List
- Task 4: Color Box Generator (bonus BOM info)
- Task 5: Form Validation (bonus confirm/prompt/alert)
